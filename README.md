## English

### Introduction
This repository contains a comprehensive collection of biology teaching materials. It is organized by subject and includes textbook chapters, lecture slides, knowledge summaries, and figures. The project also includes tools to convert PDF textbooks into Markdown format for easier processing and content generation.

### Important Notes
1. **No PDFs Included**: This repository does not contain the original textbook PDF files. Therefore, the conversion scripts will not work unless you provide the corresponding PDFs. However, the existing Typst files (`.typ`) can be directly modified.
2. **Educational Use Only**: This material is intended for learning and exchange purposes only.
3. **Compilation**: All Typst files in `knowledge/` and `slides/` directories can be compiled directly. If you find PDF versions missing for some chapters, you can compile them yourself. If Typst files are missing, please wait patiently for updates.
4. **AI Generated**: All content in this repository (summaries, slides, code) is generated by AI. Please verify the information for accuracy.

### About Typst
This project uses [Typst](https://typst.app/), a new markup-based typesetting system that is designed to be as powerful as LaTeX but much easier to learn and use. The `.typ` files in this repository are source files for the slides and knowledge summaries. You can compile them using the Typst CLI or edit them using the Typst web app or VS Code extension.

### Subjects Covered
The repository covers the following biological disciplines:
- **Animal Behavior**: Principles of animal behavior, evolution, learning, etc.
- **Biochemistry**: Chemical processes within and related to living organisms.
- **Botany**: Plant science.
- **Economy of Nature**: Ecology and environmental systems.
- **Evolution**: Evolutionary biology.
- **Human Physiology**: Functions and mechanisms of the human body.
- **Molecular Biology**: Biology at the molecular level.
- **Neurobiology**: Principles of neurobiology, neural signaling, sensory systems, and brain function.
- **Plant Physiology**: Plant physiology and growth mechanisms.
- **Zoology**: Animal biology.

### Project Structure
Each subject folder typically follows this structure:
- `chapters/`: Contains textbook chapters (PDF and converted Markdown files).
- `slides/`: Lecture slides (mostly in Typst format `.typ`).
- `knowledge/`: Knowledge summaries and notes (Typst format).
- `image_list/`: Lists of images used in materials.
- `fig/`: Figures and images.
- `output/`: Generated output files.
- `split_pdf.py`: (In some folders) Script to split large PDF textbooks into chapters.

### Workflow
1. **Split PDF**: Split the textbook PDF based on bookmarks.
2. **Generate Content**: Use the prompt in `teaching-resource.md.agent.md` to generate the teaching materials.
3. **Update Images**: Take screenshots of figures and replace the image paths in the `image_list` files.

   ```bash
   python convert_pdfs.py
   ```

## 中文

### 简介
本仓库包含一套全面的生物学教学资料。资料按学科分类，包括教材章节、讲义幻灯片、知识点总结和图表。本项目还包含将 PDF 教材转换为 Markdown 格式的工具，以便于处理和内容生成。

### 重要提示
1. **不含原始 PDF**：本项目中不包含教材原始 PDF 文件，所以脚本文件无法直接使用（除非您自行提供 PDF）。但是项目里的 Typst 文件 (`.typ`) 可以直接修改。
2. **仅供学习交流**：本项目仅用于学习交流目的。
3. **编译说明**：所有的 `knowledge` 和 `slides` 目录下的 Typst 文件都可以直接编译。如果发现某些章节的 PDF 缺失，您可以自行编译。如果没有 Typst 文件，请耐心等待更新。
4. **AI 生成**：本仓库中的所有内容（总结、幻灯片、代码）均由 AI 生成。请在使用前核实信息的准确性。

### 关于 Typst
本项目使用 [Typst](https://typst.app/) 进行排版。Typst 是一个新的基于标记的排版系统，旨在具有与 LaTeX 相当的强大功能，但更易于学习和使用。本仓库中的 `.typ` 文件是幻灯片和知识点总结的源文件。您可以使用 Typst CLI 编译它们，或使用 Typst Web 应用程序或 VS Code 扩展进行编辑。

### 涵盖学科
本仓库涵盖以下生物学科：
- **Animal Behavior (动物行为学)**：动物行为原理、进化、学习等。
- **Biochemistry (生物化学)**：生物体内的化学过程。
- **Botany (植物学)**：植物科学。
- **Economy of Nature (自然经济体系/生态学)**：生态学与环境系统。
- **Evolution (进化论)**：进化生物学。
- **Human Physiology (人体生理学)**：人体功能与机制。
- **Molecular Biology (分子生物学)**：分子水平的生物学。
- **Neurobiology (神经生物学)**：神经生物学原理、神经信号传导、感觉系统与脑功能。
- **Plant Physiology (植物生理学)**：植物生理与生长机制。
- **Zoology (动物学)**：动物生物学。

### 项目结构
每个学科文件夹通常遵循以下结构：
- `chapters/`：包含教材章节（PDF 和转换后的 Markdown 文件）。
- `slides/`：讲义幻灯片（主要为 Typst 格式 `.typ`）。
- `knowledge/`：知识点总结和笔记（Typst 格式）。
- `image_list/`：资料中使用的图片列表。
- `fig/`：图表和图片。
- `output/`：生成的输出文件。
- `split_pdf.py`：（在部分文件夹中）用于将大型 PDF 教材分割为章节的脚本。

### 工作流
1. **拆分 PDF**：根据书签拆分 PDF 文件。
2. **生成内容**：使用 `teaching-resource.md.agent.md` 中的提示词生成文件。
3. **替换图片**：截图并替换 `image_list` 文件中的图片地址。

### Human Physiology 支持
- 已新增 `Human Physiology` 目录，包含章节、知识点和幻灯片框架。
- 图片资源尚未整理完成，`image_list` 与相关插图将后续补充。

### 使用方法
将 PDF 转换为 Markdown：
1. 确保已安装所需的依赖项（例如 `markitdown`）。
2. 在根目录下运行脚本：
   ```bash
   python convert_pdfs.py
   ```
